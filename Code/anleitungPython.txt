 Befehle in PYthon zur nutzung der GPIO Pins

 Alle Befehle starten mit GPIO , danacah folgt fast immer ein Punkt un dann die Funktion die man nutzen möchte.
GPIO.setup(PIN, MODE)   stelt ein ob ich ein Pin als input oder Output nutzen möchte pin ist der Pin auf dem Rasperri
GPIO.input(PIN)         fragt den angebenen PIN ab, die Ausgabe ist ein Bool wert.
GPIO.output(PIN, ZUstand) Der angebene PIn wird je nach eingabe auf HIGH oder LOW gestellt dies wird über Bool werte
                            gemacht TRUE = HIGH und FALSE = LOW.
GPIO.cleanup() Gibt alle benutzen Pins wieder Frei
